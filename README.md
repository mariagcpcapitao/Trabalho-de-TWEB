# Trabalho de TWEB

Trabalho prático da unidade curricular **Tecnologias Web (TWEB)**.

## Tecnologias

- HTML
- SCSS / CSS
- JavaScript

## Estrutura

- `index.html` — página principal
- `estilos.scss` / `estilos.css` — estilos (compilados a partir do SCSS)
- `logica.js` — lógica/interatividade da página
- `img/` — recursos de imagem

## Como correr

1. Clona o repositório.
2. Abre `index.html` diretamente no browser, ou usa uma extensão como o Live Server (VS Code).
